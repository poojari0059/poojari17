   <b><h1>Equation System</h1></b>
<font face="arial">
<font size="4">
<p>

   Meghna is interested in solving equations. She always tries to implement new types of equations.
 This year she designed an interesting equation system.
The equation system is a set of equations, where each equation has following properties:-<br>
1. Equation will always be a valid equation.<br>
2. Equation contains  only  alphabets(a-z), operators(+,-,*) and numbers.<br>
3. The alphabets denote variables of equations.<br>
4. The maximum number of variables cannot be greater than 3 and LHS of equation can contain only 1 variable.<br>
 You have been given  <b>N</b> equations and two values which denote values of <b>a</b> and <b>b</b> respectively. Can you find the value of the other variables?<br>
 If possible, find the value of all other variables present in equation system. Assume that there will always be a unique value for the variables.<br>
You will be awarded by Meghna if you are able to answer correctly.<br>
<b>NOTE:-</b> LHS of each equation will be always unique and <b>a</b> and <b>b</b> can not be use in LHS.
</p>

<b>Input Format</b><br>
First line of the input has an integer  <b>T</b> (denoting the test cases), each test case containing :<br>
 1.<b>N</b> (number of equations),<b>a</b> and <b>b</b>(value of variables).<br>
  2.Next <b>N</b> lines contain equations(<b>E<sub>i</sub></b>).<br><br>
  

<b>Constraints</b>
<ul>
<li>1<=<b>N</b><=24</li>
<li>-10<=<b>Value of variables(a,b)</b><=10</li>
</ul>
<b>NOTE</b><br>
<ul>
  <li>only a,b,c,d,...,y,z will use as variable name.
  <li>Each input equation will given as string formt.</li>
  </ul>
<b>Output Format</b><br>
For each test case, print the values for rest of the variables separated by space according to alphabetical order of variable name, if possible to calculate value of all variables.<br>If the value of any variable can not be calculated then print <b>NA</b> in new line.<br><br>
<b>Sample Input</b> <br>
5<br>
3 2 1<br>
c=b+a<br>
e=d*c<br>
d=a-b<br>
3 1 1<br>
c=b+a<br>
e=d*c<br>
d=a-f<br>
1 2 3<br>
c=a+b<br>
3 4 3<br>
d=b*c<br>
e=d*a<br>
c=a-b<br>
2 3 -10<br>
f=-d-a<br>
d=a<br>
<b>Sample output</b><br>
3 1 3<br>
NA<br>
5<br>
1 3 12<br>
3 -6<br><br>

<b>Explanation</b><br>

Test Case #00:<h4 style="color:red;">As given number of equations=3 , a=2 and b=1 we find c=3 , d=1 and e=3 by substituting a and b in the above eqations.</h4><br>
Test Case #01:<h4 style="color:red;">As given number of equations=3 , a=1 and b=1 we find c=2 ,d can't be determined because f can't be found,since d can't be determind so we can't find the value of e also.</h4><br>
